# 小米运动自动刷步数

> 小米运动自动刷步数

## Github Actions 部署指南

### 一、Fork 此仓库

### 二、设置账号密码

添加名为  **XMYD_USER**、**XMYD_PASSWD**、**XMYD_STEP**、**TG_BOT_TOKEN**、**TG_USER_ID** 的变量，值分别为 、 **账号（仅支持手机号）**、**密码**、**步数（0则为1w-2w之间随机 或自定义随机范围[18000-25000]）**

推送为TG推送

> Settings-->Secrets-->New secret

### 三、多账户(用不上请忽略)

多账户请用 **#** 分割
